
# aruResearch

El paquete `aruResearch` provee plantillas de RMarkdown para reportes
técnicos de la Fundación Aru.

## Plantillas incluidas

- `arubeamer`: Para una presentación en beamer en un formato
  institucional.
- `arubrief`: Para generar un Brief institucional.
- `arusearch`: Para generar una salida para la revista Aru Search.
- `arunote`: Para notas metodológicas

## Instalación

Para instalar la versión de desarrollo desde GitHub:

``` r
remotes::install_github("AlvaroLimber/aruResearch")
```
